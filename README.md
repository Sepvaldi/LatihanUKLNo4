# LatihanUKLNo4

Program ini berfungsi untuk menghitung volume tabung berdasarkan jari-jari alas dan tinggi yang dimasukkan oleh pengguna.

Program diawali dengan menulis Scanner agar bisa menerima input dari pengguna.
Program menampilkan judul dan meminta pengguna untuk memasukkan nilai jari-jari serta tinggi tabung.
Nilai-nilai tersebut disimpan dalam variabel dan untuk nilai ğœ‹ sudah ditulis di dalam program yaitu 3,14.
Program menghitung volume tabung menggunakan rumus ğœ‹ * r * r * t
Hasil perhitungan kemudian ditampilkan di layar sebagai output.
